## Installation

```sh
npm install tslint-config-pedantic --save-dev
```


--------------------------------------------------------------------------------

## Usage

Extend your `tsling.json`:

**Common**:

```json
{
  "extends": [
    "tslint-config-pedantic"
  ]
}
```


**Angular.js (1.x)**:

```json
{
  "extends": [
    "tslint-config-pedantic/angularjs"
  ]
}
```
